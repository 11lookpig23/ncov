# 开发

基于Vue 3 + Echart搭建的可视化大屏。

目前vue 3生态未成熟，无法使用高阶技能~

## 技术栈/Technical framework

- vue 3.0.0-alpha
- echarts
- webpack
- AI大数据框架：[Tensorflow](https://tensorflow.google.cn/)
